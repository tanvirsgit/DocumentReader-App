System Features


In this project, a camera and eye tracking software APIs are utilized to generate eye detection and tracking algorithms. The algorithms after being fed data from the multiple platforms will be able to generate a statistic and plot a summary of information relevant to that current user and report it to an admin in a clear, concise and presentable way making use of charts and graphs to aid readability.

Fig: System Diagram

4.1 PDF Reader


4.1.1 Description and Priority

The PDF Reader application stores user information by creating an account via Facebook/Gmail or user provided credentials. The PDFs saved with the software is secured and only granted access after a successful login.The data stored has a low risk of being sabotaged by adversaries, as the server validates rigorous checks to requesting accounts each time the app is opened. The other risk this type of application might have is the the radiation from an always active camera directed at the pupils, but that concern is mitigated by studies showing that it is relatively risk free as long as the user is careful with their usage.


4.1.2 Stimulus/Response Sequences

  • Open account
  • Open PDF
  • Read the PDF
  • Save PDF locally for instant access and in online for switching platform access(Eg. Between Mobile and Computer)
  • Send User statistics to server for processing
  • Send email to admins

4.1.3 Functional Requirements

Admin: Admin will monitor the whole system and the communication between server user. If there is a system failure admin, all operations will be halted and accounts suspended until it is fixed. Admins have the right to revoke accounts, edit account information, and get emails with reports of user progress and view statistics.

Server: The server is responsible for communication between the different platforms and process reports using image analysis with data gathered the mobile device and PCs. It will act as the bridge between these different machines and keep confidential information.

User: Users can create accounts within the PDF reader. They can open a PDF, save a PDF, and view their dashboard.
